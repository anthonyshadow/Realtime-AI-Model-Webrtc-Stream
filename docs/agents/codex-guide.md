# Codex Guide

Use this for Codex-specific working habits in this repo.

## Start

- Read root [../../AGENTS.md](../../AGENTS.md).
- Read [always-read.md](always-read.md).

## While Editing

- Prefer `rg` for search.
- Use the existing source boundary.
- Keep docs and source scaffolding conventions in sync.
- Run focused checks before broad checks.

## Finish

- Summarize changed docs/source scaffolding.
- Report checks run and any checks not run.
