# Skill Template
> Last updated: 2026-06-28

Copy this structure when adding a skill.

```md
# Skill: [Name]
> Last updated: YYYY-MM-DD

## Use when

- Short list of situations where this skill applies.

## Read first

- Minimal required docs.
- Minimal required source areas.

## Steps

1. Concrete step.
2. Concrete step.
3. Concrete step.

## Checks

- Commands or validations to run.

## Do not

- Common mistakes to avoid.

## Update docs

- Which docs should be updated if implementation changes.
- Update the Last updated date of any document changed.
```
