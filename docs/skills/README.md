# Skills
> Last updated: 2026-06-28

Skills are reusable task procedures for this repo. They are not general documentation.

Use [skill-index.md](skill-index.md) to choose a skill. Use [skill-template.md](skill-template.md) when adding a new one.

## Rules

- Keep skills short, procedural, and task-specific.
- Link to the minimal docs and source areas needed.
- Do not duplicate architecture docs.
- Include checks and common mistakes.
- Update `Last updated` when a skill changes.
